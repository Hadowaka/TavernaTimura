# 🍺 Таверна Timura — проект игрового сайта и форума
## 📌 Что это?
*Таверна Timura* — это полноценный игровой портал (веб-приложение), где пользователи могут:
- Искать и просматривать каталог игр
- Ставить оценки (от 1 до 10) и писать комментарии
- Общаться на форуме (создавать темы и отвечать)
- Читать новостную ленту игровой индустрии
- Управлять личным кабинетом (аватар, настройки, оценки)
