📚 What is this?

This is a minimal frontend for a FastAPI forum API. The real learning happens in the Python files. Every button here fires an HTTP request to the API — watch the Request / Response Log below to see exactly what is sent and received.

You can also use the interactive /docs page or a tool like curl / Postman to test the API directly.

GET /posts — list all posts
GET /posts/{id} — one post
POST /posts — create post
PUT /posts/{id} — update post
DELETE /posts/{id} — delete post
POST /comments/post/{id} — add comment

POST /posts Create a New Post
